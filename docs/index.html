üèóÔ∏è Multi-Agent Architecture

The system implements a multi-agent architecture pattern where each agent has specific responsibilities:

  2. MasterAgent orchestrates the investigation with strategic planning
  3. InvestigatorAgent executes anomaly detection using ML algorithms
  4. AnalystAgent performs correlation analysis and pattern identification

  6. ContextMemoryAgent maintains context between interactions
