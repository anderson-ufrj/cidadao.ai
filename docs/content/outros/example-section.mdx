# Exemplo: Como Usar MDX Mantendo Seu HTML

Este é um exemplo de como você pode **misturar Markdown com seu HTML customizado**.

## Você pode usar Markdown normal:

- Lista de itens
- **Texto em negrito**
- *Texto em itálico*
- [Links](https://example.com)

## E manter seu HTML exato:

<div style="background-color: var(--bg-tertiary); border: 1px solid var(--border); border-radius: 0.5rem; padding: 2rem; margin: 2rem 0; font-family: 'Inter', sans-serif; line-height: 1.8;">
    <h3 style="font-size: 1.25rem; font-weight: 700; margin-bottom: 1rem; color: var(--text-primary); text-align: center;">Resumo Acadêmico</h3>
    <p style="font-size: 1rem; color: var(--text-secondary); text-align: justify;">
        Seu HTML customizado funciona EXATAMENTE como antes. Todas as variáveis CSS, estilos inline, estrutura - tudo preservado.
    </p>
</div>

## Fórmulas matemáticas continuam funcionando:

A probabilidade condicional é definida como:

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$
